# Gym Management System

A comprehensive gym management solution with a FastAPI backend and a React frontend.

## Features
- **Member Management**: Track member details, subscriptions, and progress.
- **Coach Directory**: Manage professional trainers and their specialized packages.
- **Branch Management**: Overview of multiple gym locations.
- **Finance & Analytics**: Real-time revenue tracking and system reports.
- **Interactive Chat**: Real-time communication between coaches and members.

## Project Structure
- `backend/`: FastAPI application.
- `frontend/`: React application built with Vite and Tailwind CSS.

## Getting Started
Refer to the individual `README.md` files in the `backend/` and `frontend/` directories for detailed setup instructions.
